## About The Project

Ashish Techfolio is a modern developer portfolio focused on:

- Backend Development
- Full Stack Web Applications
- FastAPI & Flask APIs
- PostgreSQL Integration
- Generative AI & RAG Systems

The portfolio dynamically fetches selected GitHub repositories using the GitHub API and showcases projects in a responsive UI with dark/light mode support.
